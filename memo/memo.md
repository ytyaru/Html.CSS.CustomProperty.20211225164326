# ユーザ設定項目

* 縦書き／横書き
* フォント
	* サイズ
	* ファミリ（等幅／プロポーショナル。ゴシック体／明朝体）
	* 太さ(100〜700)
* 字間
* 行間
* 色（文字色／背景色）
* スライド／ページ捲り
* 次／前
* トップ／続きから

```
縦｜横
40  字／行 |---------■-------|
0.5 字間
0.5 行間
[] 背景色 [] 文字色
[フォント]
```

## 色

* 文字色（白／黒）
* 背景色（白または黒とコントラスト比が4.5以上の色。その範囲内においてLChで色を指定する）
